# Article Tuwen — 素材转图文一条龙

> One-shot pipeline: raw material → 4000-word article → 5-9 social cards + text summary

## 这是什么

Article Tuwen 是一个素材转图文管线：丢进去 URL、文件或文本，它帮你写长文、排版渲染、截图出 5-9 张 1080×1440 社交卡片，外加 800-1000 字压缩文字稿。

**架构定位**：编排层，自身不执行内容生产，而是调度两个子技能：
- **transcript-crafter** — 长文撰写（10维度提取 + 5工具搜索补充 + 事实核查）
- **xhs-crafter** — 排版渲染（HTML排版 + 截图 + 文字稿压缩）

## 数据处理与隐私

- **本地处理优先**：所有素材获取、长文撰写、排版渲染、截图均在本地完成
- **云上传需确认**：飞书云盘同步是可选步骤，执行前必须征得用户明确同意
- **敏感内容警告**：如果素材包含敏感或机密信息，会提醒用户云上传风险
- **不传输原始素材**：飞书云盘仅上传生成的 PNG 和文字稿，不上传原始素材文件

## 触发方式

在 AI 编码助手中说：

```
转写成图文
```

然后粘贴 URL、文件路径或文本。

**注意**：触发后会自动处理素材并生成本地文件。如需同步到飞书云盘，会在上传前询问确认。

### 触发边界

- **触发**：用户明确说"转写成图文"或"转换成图文" + 提供素材
- **不触发**：只说"写文章"/"排版"而无"图文"关键词；只提供URL但无转换意图

## 输出

```
桌面\<slug>公众号配图\
├── p1.png ~ pN.png        # 5-9张 (1080×1440)
├── <slug>-article.md       # 长文源文件
├── <slug>-文字稿.txt       # 800-1000字
└── used-images.json        # 图片来源记录
```

飞书云盘同步为可选项，需用户确认后执行。

## 核心特性

- **本地零确认，云上传需确认**：Phase 1-3 全自动，Phase 4 云上传前询问
- **多样性轮换**：8封面词 + 8封底词 + 5主题池，每次运行不重复
- **截图安全**：仅清理 8090 端口占用进程，不影响其他 Python 工作
- **字号铁律**：正文≥32px，辅助≥28px，元信息≥24px
- **密度铁律**：活跃构图≥78%画布高度
